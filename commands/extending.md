---
description: Guide for adding skills, hooks, agents, MCPs, and GitHub Actions to ~/.claude
---

Read `~/.claude/EXTENDING.md` and follow its conventions for the requested task. This document is the source of truth for:

- **Custom skills** — creating commands in `commands/`
- **Hooks** — automation triggers
- **GitHub Actions** — scheduled tasks
- **MCP server configurations**
- **Agent definitions** — creating and maintaining agents in `agents/`

After reading EXTENDING.md, help the user create or update the requested component following the documented templates and conventions.
